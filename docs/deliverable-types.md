## Deliverable Types

- **Proof of Concept** — feasibility validation
- **Prototype / Wireframes** — interactive mockups
- **MVP** — minimum viable product, smallest shippable version
- **Beta** — feature-complete, limited audience
- **v1.0 / Launch** — full public release
- **Technical Spec / ADRs** — design decisions and rationale
- **API Specification** — interface contracts
- **Design System** — reusable component library and patterns
- **Documentation** — user guides, admin manuals
- **Migration Plan** — if replacing existing systems
- **Test Plan / QA Strategy** — testing approach and coverage
- **Deployment / Rollout Plan** — release strategy and infrastructure
- **Analytics / Monitoring** — observability and metrics setup
- **Training Materials** — onboarding and user training
